# password-generator

_v0.0.1_

### Description

A simple JavaScript file that exports a _passwordGenerator_ function.
She returns a random _String_ with :

-   8 characters _min_
-   1 number _min_
-   1 uppercase _min_
-   1 lowercase _min_
-   1 symbol _min_

| Option | Description                  | Default value |
| ------ | ---------------------------- | ------------- |
| length | Integer, length of password. | 12            |
